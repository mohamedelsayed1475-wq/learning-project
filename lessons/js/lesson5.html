📦 المصفوفات والكائنات (Arrays & Objects)

تعلم كيف تخزن وتتحكم في مجموعات البيانات المعقدة والمرتبطة ببعضها البعض في جافاسكريبت باستخدام المصفوفات والكائنات!

📦 المصفوفات (Arrays)

المصفوفة هي قائمة مرتبة من البيانات. يمكن للمصفوفة أن تحتوي على أي نوع من البيانات (أرقام، نصوص، كائنات، إلخ). 

// 1. تعريف مصفوفة جديدة
const fruits = ['تفاح', 'موز', 'برتقال'];

// 2. الوصول لعناصر المصفوفة باستخدام الفهرس (Index) - يبدأ من 0
console.log(fruits[0]); // تفاح
console.log(fruits[1]); // موز
console.log(fruits.length); // 3 (عدد العناصر)

// 3. إضافة وحذف عناصر من أطراف المصفوفة
fruits.push('مانجو');     // إضافة عنصر جديد في النهاية
console.log(fruits);      // ['تفاح', 'موز', 'برتقال', 'مانجو']

const lastFruit = fruits.pop(); // حذف وإرجاع آخر عنصر
console.log(lastFruit);   // مانجو
console.log(fruits);      // ['تفاح', 'موز', 'برتقال']

fruits.unshift('كرز');    // إضافة عنصر في البداية
console.log(fruits);      // ['كرز', 'تفاح', 'موز', 'برتقال']

fruits.shift();           // حذف أول عنصر
console.log(fruits);      // ['تفاح', 'موز', 'برتقال']

🔄 دوال التكرار والتحول الحديثة (Array Methods)

💡 المصفوفات في جافاسكريبت تمتلك دوالاً قوية ومتقدمة تغنينا عن استخدام الحلقات التكرارية for التقليدية في كثير من الأحيان.

const numbers = [1, 2, 3, 4, 5];

// 1️⃣ forEach: المرور على كل عنصر لتنفيذ عملية معينة
numbers.forEach(function(num) {
  console.log(num * 2); // يطبع: 2, 4, 6, 8, 10
});

// 2️⃣ map: إنشاء مصفوفة جديدة معدلة بناءً على المصفوفة الأصلية
const doubled = numbers.map(num => num * 2);
console.log(doubled); // [2, 4, 6, 8, 10]

// 3️⃣ filter: تصفية المصفوفة وإرجاع العناصر التي تحقق الشرط فقط
const evens = numbers.filter(num => num % 2 === 0);
console.log(evens); // [2, 4]

📂 الكائنات (Objects)

الكائن هو هيكل بيانات غير مرتب يخزن البيانات في صورة مفتاح وقيمة (Key - Value Pairs). يمثل كياناً حقيقياً مثل شخص أو منتج. 

// 1. تعريف الكائن (Object Literal)
const user = {
  firstName: 'أحمد',
  age: 25,
  isStudent: false,
  skills: ['HTML', 'CSS', 'JS'],
  // دالة داخل الكائن (تسمى Method)
  sayHello: function() {
    return `مرحباً، أنا اسمي ${this.firstName}!`;
  }
};

// 2. الوصول للبيانات بطريقة النقطة (Dot Notation) ✅ الأكثر شيوعاً
console.log(user.firstName); // أحمد
console.log(user.skills[0]); // HTML
console.log(user.sayHello()); // مرحباً، أنا اسمي أحمد!

// 3. الوصول بطريقة الأقواس المربعة (Bracket Notation)
console.log(user['age']); // 25
// مفيدة عندما يكون اسم المفتاح مخزناً داخل متغير:
const keyToGet = 'isStudent';
console.log(user[keyToGet]); // false

// 4. تعديل وإضافة الخصائص
user.age = 26;            // تعديل قيمة موجودة
user.country = 'مصر';      // إضافة خاصية جديدة تماماً

👥 مصفوفة الكائنات (Array of Objects)

في المشاريع الحقيقية، تأتي البيانات دائماً على شكل مصفوفة تحتوي على كائنات متعددة (مثل قائمة المستخدمين أو المنتجات). 

const students = [
  { id: 1, name: 'سارة', score: 95 },
  { id: 2, name: 'محمد', score: 78 },
  { id: 3, name: 'خالد', score: 88 }
];

// الوصول لاسم الطالب الأول
console.log(students[0].name); // سارة

// تصفية الطلاب الحاصلين على درجة أكبر من 80
const topStudents = students.filter(student => student.score > 80);
console.log(topStudents);
/*
  [
    { id: 1, name: 'سارة', score: 95 },
    { id: 3, name: 'خالد', score: 88 }
  ]
*/

// الحصول على مصفوفة تحتوي على الأسماء فقط
const names = students.map(student => student.name);
console.log(names); // ['سارة', 'محمد', 'خالد']

🎓 ما تعلمناه في هذا الدرس

  • ✅ المصفوفات قوائم مرتبة تبدأ من الفهرس 0.
  • ✅ push و pop للتعامل مع نهاية المصفوفة.
  • ✅ دوال map و filter و forEach لتكرار ومعالجة المصفوفات بسلاسة.
  • ✅ الكائنات تخزن البيانات في شكل (مفتاح وقيمة).
  • ✅ نصل لخصائص الكائنات عبر Dot Notation أو Bracket Notation.
  • ✅ دمج المصفوفات مع الكائنات يمثل الهيكل الأساسي للبيانات في الويب.
