⚡ مقدمة في JavaScript

JavaScript هي لغة البرمجة التي تُحيي صفحات الويب وتجعلها تفاعلية! ستتعلم في هذا الدرس ما هي JS وكيف تضيفها لصفحاتك وكيف تطبع بياناتك.

🤔 ما هي JavaScript؟

🧠 JavaScript هي لغة البرمجة الوحيدة التي يفهمها المتصفح مباشرة!

  • 🟠 HTML = هيكل الصفحة (ماذا يوجد؟)
  • 🔵 CSS = شكل الصفحة (كيف تبدو؟)
  • 🟡 JavaScript = سلوك الصفحة (ماذا تفعل؟)

مثال: زر يفتح قائمة، نموذج يتحقق من البيانات، بيانات تُحمَّل بدون تحديث الصفحة!

📝 كيف تضيف JavaScript؟

1. داخلي مباشر (Inline)

<!-- مباشرة في سمات HTML - لا يُنصح به -->
<button onclick="alert('مرحباً!')">اضغط هنا</button>

<!--
  onclick = حدث النقر
  alert() = دالة تعرض نافذة منبثقة
  ⚠️ يخلط HTML مع JS - غير مُنصح به
-->

2. في وسم script (Internal)

<script>
  // هنا نكتب JavaScript مباشرة في ملف HTML
  // التعليق في JS يبدأ بـ //

  console.log('مرحباً بالعالم!');
  // console.log = طباعة في وحدة تحكم المتصفح

  alert('أهلاً بك!');
  // alert = نافذة منبثقة
</script>

<!--
  ملاحظة مهمة: ضع <script> قبل </body>
  مش في <head>
  لأن HTML لازم يتحمل الأول قبل ما JS تبدأ تعمل
-->

3. ملف خارجي (External) ✅ الأفضل

<!-- في ملف HTML قبل </body> -->
<script src="script.js"></script>

<!--
  src = مسار ملف JavaScript
  ✅ أفضل للمشاريع الكبيرة لأن:
  - فصل JS عن HTML
  - إعادة استخدام نفس الملف
  - أسهل في التعديل
-->

🖨️ طرق إخراج البيانات

// 1. طباعة في Console المتصفح (للمطورين فقط)
console.log('مرحباً!');
// مهم جداً للـ Debugging - اكتشاف الأخطاء

console.log(42);
console.log(true);
console.log([1, 2, 3]);
// يمكن طباعة أي نوع بيانات

console.error('حدث خطأ!');
// يطبع رسالة خطأ باللون الأحمر

console.warn('تحذير!');
// يطبع تحذير باللون الأصفر

// 2. نافذة منبثقة (تُزعج المستخدم - تجنبها)
alert('هذه رسالة تنبيه!');

// 3. نافذة تأكيد - تُعيد true أو false
const confirmed = confirm('هل أنت متأكد؟');
console.log(confirmed); // true أو false

// 4. نافذة إدخال بيانات
const name = prompt('ما اسمك؟');
console.log('مرحباً ' + name);

// 5. الكتابة في صفحة HTML (لا يُنصح به)
document.write('<h1>مرحباً!</h1>');
// يحذف محتوى الصفحة ويضع هذا بدلاً منه!

💬 التعليقات في JavaScript

// هذا تعليق على سطر واحد
// كل ما بعد // في نفس السطر = تعليق
// لا يُنفَّذ ولا يظهر للمستخدم

/*
  هذا تعليق متعدد الأسطر
  يمكن امتداده على أكثر من سطر
  مفيد لشرح أجزاء كبيرة من الكود
*/

// ✅ اكتب تعليقات وافية لكودك!
// الكود الجيد هو الذي يشرح نفسه
// لكن التعليقات تشرح السبب (WHY) وليس الماذا (WHAT)

const price = 100; // سعر المنتج بالجنيه
const discount = 0.2; // خصم 20%
const finalPrice = price * (1 - discount);
// السعر النهائي = 80 جنيه
