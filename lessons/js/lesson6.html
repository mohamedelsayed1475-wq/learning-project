🌳 التحكم في عناصر الصفحة (DOM Manipulation)

تعديل محتوى الصفحة بشكل حيوي وديناميكي! تعلم كيف تستهدف عناصر HTML وتغير ألوانها، نصوصها، وتنسيقاتها، بل وتصنع عناصر جديدة تماماً من خلال كود جافاسكريبت.

🌳 ما هي شجرة مستند الويب (DOM)؟

DOM اختصار لـ Document Object Model (نموذج كائن المستند). عندما يقوم المتصفح بتحميل صفحة HTML، فإنه يقوم ببناء شجرة تمثل الصفحة، حيث يمثل كل وسم أو عنصر في الصفحة "عقدة" (Node) في هذه الشجرة.

💡 يمثل كائن document البوابة الرئيسية للتحكم في هذه الشجرة. من خلاله نستطيع الوصول لكل شيء داخل الصفحة وتعديله!

🔍 تحديد وتحديد العناصر (Selecting Elements)

قبل تعديل أي عنصر في الصفحة، يجب أولاً العثور عليه وتحديده بداخل كود جافاسكريبت:

// 1️⃣ تحديد عنصر عن طريق الـ ID الخاص به
const title = document.getElementById('main-title');

// 2️⃣ تحديد عنصر باستخدام محددات CSS (الأحدث والأكثر مرونة) ✅
const logo = document.querySelector('.logo-text');  // يستهدف أول عنصر يحمل كلاس logo-text
const firstParagraph = document.querySelector('p'); // يستهدف أول وسم <p> في الصفحة
const activeLink = document.querySelector('nav a.active'); // محدد مركب مثل CSS

// 3️⃣ تحديد عناصر متعددة من نفس النوع
const allButtons = document.querySelectorAll('.btn'); // يعيد قائمة (NodeList) بجميع الأزرار
allButtons.forEach(btn => {
  // يمكننا المرور على كل عنصر على حدة
  console.log(btn);
});

✍️ تعديل المحتوى والخصائص (Modifying Elements)

بمجرد تحديد العنصر، يمكنك تغيير محتواه، تنسيقه، أو سماته:

const myDiv = document.querySelector('.card');

// 1️⃣ تعديل النص الداخلي للعنصر (بأمان)
myDiv.textContent = 'مرحباً بالمتعلمين الجدد!';

// 2️⃣ تعديل محتوى HTML الداخلي (يقبل وسوم HTML)
myDiv.innerHTML = '<h3>عنوان جديد</h3><p>فقرة جديدة</p>';

// 3️⃣ تعديل التنسيقات المباشرة (Inline Styles)
myDiv.style.color = '#ef4444'; // تغيير اللون للأحمر
myDiv.style.backgroundColor = '#fef2f2'; // لاحظ كتابة camelCase للخصائص ذات المقطعين (background-color في CSS)
myDiv.style.fontSize = '18px';

// 4️⃣ تعديل سمات العنصر (Attributes)
const image = document.querySelector('img');
image.setAttribute('src', 'new-image.jpg'); // تغيير مسار الصورة
console.log(image.getAttribute('src'));     // قراءة قيمة السمة src

🏷️ التحكم في الكلاسات (ClassList API) ✅ الأفضل للمظهر

⚠️ بدلاً من تعديل التنسيقات مباشرة بالـ Style، الممارسة الأفضل هي وضع التنسيق في كلاس بملف CSS، ثم إضافة أو حذف الكلاس باستخدام جافاسكريبت.

const box = document.querySelector('.box');

// إضافة كلاس للعنصر
box.classList.add('highlight');

// إزالة كلاس
box.classList.remove('active');

// تبديل الكلاس (إذا كان موجوداً يحذفه، وإذا لم يكن يضيفه)
box.classList.toggle('dark-mode');

// التحقق من وجود الكلاس (يرجع true أو false)
if (box.classList.contains('hidden')) {
  console.log('الصندوق مخفي حالياً');
}

➕ إنشاء وحذف العناصر ديناميكياً (Creating & Removing Elements)

// 1️⃣ إنشاء عنصر جديد في الذاكرة
const newParagraph = document.createElement('p');

// 2️⃣ تخصيص العنصر وتعبئة بياناته
newParagraph.textContent = 'تم إنشاء هذا النص بواسطة جافاسكريبت!';
newParagraph.classList.add('info-text');

// 3️⃣ إدراج العنصر داخل الصفحة (حقنه في عنصر أب)
const container = document.querySelector('.container');
container.appendChild(newParagraph); // يضاف كآخر ابن للعنصر container

// 4️⃣ حذف العنصر نهائياً من الصفحة
const oldTitle = document.getElementById('obsolete-title');
if (oldTitle) {
  oldTitle.remove(); // يحذفه فوراً من شجرة الـ DOM
}

🎓 ما تعلمناه في هذا الدرس

  • ✅ الـ DOM هو تمثيل شجري لصفحة الويب بداخل الذاكرة.
  • ✅ كائن document هو المسؤول الرئيسي عن عمليات الـ DOM.
  • ✅ نحدد العناصر باستخدام querySelector و querySelectorAll.
  • ✅ نعدل النصوص بـ textContent ووسوم HTML بـ innerHTML.
  • ✅ نغير المظهر بأمان واحترافية من خلال التحكم في الكلاسات بـ classList.
  • ✅ ننشئ عناصر جديدة بـ createElement ونلحقها بالصفحة بـ appendChild.
