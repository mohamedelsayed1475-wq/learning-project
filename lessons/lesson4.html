📋 القوائم في HTML

القوائم من أكثر عناصر HTML استخداماً في المواقع. سنتعلم ثلاثة أنواع: القوائم غير المرتبة (ul)، والقوائم المرتبة (ol)، وقوائم التعريف (dl).

1. القائمة غير المرتبة <ul>

تُستخدم <ul> (Unordered List) لعرض قائمة من العناصر بدون ترتيب معين (تظهر كنقاط). كل عنصر داخلها يكتب بوسم <li> (List Item). 

<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>

👁️ النتيجة في المتصفح:

  • HTML
  • CSS
  • JavaScript

2. القائمة المرتبة <ol>

تُستخدم <ol> (Ordered List) لعرض قائمة مرقّمة بترتيب رقمي أو أبجدي معين. مثالية لخطوات الطبخ أو التعليمات التسلسلية. 

<ol>
  <li>افتح محرر النصوص</li>
  <li>اكتب كود HTML</li>
  <li>احفظ الملف بامتداد .html</li>
  <li>افتح الملف في المتصفح</li>
</ol>

👁️ النتيجة في المتصفح:

  1. افتح محرر النصوص
  2. اكتب كود HTML
  3. احفظ الملف بامتداد .html
  4. افتح الملف في المتصفح
💡 خصائص <ol> المفيدة:
• type="A" — ترقيم بالحروف الإنجليزية الكبيرة (A, B, C)
• type="I" — ترقيم بالأرقام الرومانية (I, II, III)
• start="5" — بدء التعداد والترقيم من رقم محدد (مثل البدء من 5 بدلاً من 1)

3. القوائم المتداخلة (Nested Lists)

يمكن وضع قائمة كاملة جديدة داخل عنصر <li> لإنشاء قائمة فرعية متداخلة:

<ul>
  <li>تطوير الويب
    <ul>
      <li>HTML</li>
      <li>CSS</li>
      <li>JavaScript</li>
    </ul>
  </li>
  <li>تطوير الموبايل</li>
  <li>الذكاء الاصطناعي</li>
</ul>

👁️ النتيجة:

  • تطوير الويب
    • HTML
    • CSS
    • JavaScript
  • تطوير الموبايل
  • الذكاء الاصطناعي

4. قائمة التعريف <dl>

تُستخدم <dl> (Description List) لعرض مصطلحات يتبعها شروحات أو تعريفات لكل مصطلح. تتكون من جزئين:

  • <dt> (Description Term) — المصطلح أو الكلمة المراد تعريفها.
  • <dd> (Description Details / Definition) — التفاصيل والشرح لهذا المصطلح.
<dl>
  <dt>HTML</dt>
  <dd>لغة ترميز النصوص التشعبية، تُستخدم لبناء هيكل صفحات الويب.</dd>

  <dt>CSS</dt>
  <dd>صحائف الأنماط المتتالية، تُستخدم لتنسيق وتصميم صفحات الويب.</dd>
</dl>
