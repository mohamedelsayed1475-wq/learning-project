🎨 مقدمة وأساسيات CSS

تعلم ما هي CSS وكيف تستخدمها لتحويل صفحات HTML العادية إلى تصاميم احترافية ورائعة! ستتعلم في هذا الدرس طرق كتابة CSS والمحددات الأساسية.

🤔 ما هي CSS؟

CSS اختصار لـ Cascading Style Sheets أي "أوراق الأنماط المتتالية". هي اللغة المسؤولة عن الشكل والمظهر لصفحات الويب.

🎭 تخيل صفحة HTML كهيكل عظمي للجسم، و CSS هي الملابس والمظهر الخارجي!

  • 🟠 HTML = هيكل الصفحة (العظام والمحتوى)
  • 🔵 CSS = التصميم والمظهر (الألوان والخطوط والتخطيط)
  • 🟡 JavaScript = التفاعل والحركة (العضلات والحركة)

✍️ طرق كتابة CSS

في CSS ثلاث طرق لتطبيق التنسيق على العناصر:

1️⃣ CSS داخلي (Inline CSS)


<p style="color: red; font-size: 18px;">هذا نص أحمر</p>

2️⃣ CSS داخلي في الملف (Internal CSS)

<head>
  <style>
    /* هنا نكتب CSS بين وسم style */
    p {
      color: blue;     /* لون النص أزرق */
      font-size: 16px; /* حجم الخط 16 بكسل */
    }
  </style>
</head>

3️⃣ CSS خارجي (External CSS) ✅ الأفضل


<head>
  <link rel="stylesheet" href="style.css">
</head>

🎯 بناء قاعدة CSS (CSS Rule)

كل قاعدة CSS تتكون من ثلاثة أجزاء:

/*
  بناء قاعدة CSS الأساسية:

  المحدد { الخاصية: القيمة; }

  Selector { property: value; }
*/

p {
  /* المحدد p = يستهدف جميع عناصر <p> في الصفحة */

  color: darkblue;
  /* color = اسم الخاصية */
  /* darkblue = القيمة */
  /* ; = الفاصلة المنقوطة تنهي كل خاصية */

  font-size: 18px;
  /* يمكن كتابة خصائص متعددة في نفس المحدد */

  line-height: 1.6;
  /* ارتفاع السطر = 1.6 ضعف حجم الخط */
}

/*
  ملاحظة: التعليقات في CSS بين /* و *-/
  (أزلت النجمة الأخيرة لتجنب إغلاق التعليق مبكراً)
  أي سطر بين العلامتين = تعليق لا يُنفَّذ
*/

🎯 المحددات الأساسية (Selectors)

1. محدد العنصر (Element Selector)

/* يستهدف جميع عناصر من نفس النوع */
h1 {
  color: purple;   /* كل h1 في الصفحة هيبقى لونه بنفسجي */
}

p {
  font-size: 16px; /* كل p في الصفحة حجمها 16px */
}

button {
  background: blue; /* كل الأزرار خلفيتها زرقاء */
}

2. محدد الكلاس (Class Selector) - الأكثر استخداماً

/* في HTML تعطي العنصر class */
<!-- <p class="intro-text">نص المقدمة</p> -->

/* في CSS تستهدفه بالنقطة . */
.intro-text {
  color: #2563eb;  /* لون أزرق HEX */
  font-weight: bold; /* خط عريض */
}

/*
  ✅ مميزات الكلاس:
  - يمكن تطبيقه على عناصر متعددة
  - عنصر واحد يمكنه امتلاك كلاسات متعددة
  - مثال: <div class="card featured highlight">
*/

3. محدد الـ ID (ID Selector)

/* في HTML تعطي العنصر id فريد */
<!-- <header id="main-header">...</header> -->

/* في CSS تستهدفه بـ # */
#main-header {
  background: #1e293b; /* لون خلفية داكن */
  padding: 20px;       /* مسافة داخلية 20 بكسل */
}

/*
  ⚠️ قواعد الـ ID:
  - يجب أن يكون فريداً في الصفحة (لا يتكرر)
  - خصوصيته (Specificity) أعلى من الكلاس
  - يُنصح باستخدام الكلاس بدلاً منه في CSS
  - الـ ID أفضل للـ JavaScript
*/

4. المحدد الشامل (Universal Selector)

/* * تعني "كل العناصر في الصفحة" */
* {
  box-sizing: border-box;
  /* هذا الإعداد مهم جداً - يجعل حساب الأبعاد أسهل */
  /* سنشرحه بالتفصيل في درس Box Model */

  margin: 0;
  padding: 0;
  /* إزالة الهوامش الافتراضية من كل العناصر */
  /* ممارسة شائعة في بداية كل مشروع CSS */
}

5. المحدد المتعدد (Multiple Selectors)

/* تستهدف عناصر متعددة بنفس النمط */
h1, h2, h3 {
  font-family: 'Cairo', sans-serif;
  /* تطبق نفس الخط على h1 وh2 وh3 */
  /* الفاصلة تفصل بين المحددات */
}

.btn-primary, .btn-secondary {
  border-radius: 8px;
  padding: 10px 20px;
  /* تطبق نفس التنسيق على نوعي الأزرار */
}

6. محدد النسب (Descendant Selector)

/* يستهدف العنصر داخل عنصر آخر */
nav a {
  color: white;
  /* يستهدف كل روابط <a> الموجودة داخل <nav> فقط */
  /* وليس كل روابط الصفحة */
}

.card .title {
  font-size: 20px;
  /* يستهدف كل عنصر بكلاس .title داخل عنصر بكلاس .card */
}

⚖️ أولوية CSS (Specificity)

عندما تتعارض قواعد CSS، أيها يفوز؟

/*
  ترتيب الأولوية من الأقل للأعلى:

  1️⃣ المحدد الشامل (*) = 0 نقطة
  2️⃣ محدد العنصر (p, h1) = 1 نقطة
  3️⃣ محدد الكلاس (.card) = 10 نقطة
  4️⃣ محدد الـ ID (#header) = 100 نقطة
  5️⃣ Inline style = 1000 نقطة
  6️⃣ !important = يتغلب على الكل ⚠️
*/

p {
  color: red;    /* 1 نقطة */
}

.text {
  color: blue;   /* 10 نقاط - يفوز على p */
}

#intro {
  color: green;  /* 100 نقطة - يفوز على الكلاس */
}

/*
  <p id="intro" class="text">ما لون هذا النص؟</p>
  الجواب: أخضر! لأن #intro (100 نقطة) يفوز

  ⚠️ تجنب استخدام !important قدر المستطاع
  لأنه يجعل التعديلات مستقبلاً أصعب
*/

🎓 ما تعلمناه في هذا الدرس

  • ✅ CSS تعني Cascading Style Sheets وتتحكم في مظهر الصفحة
  • ✅ ثلاث طرق لكتابة CSS: Inline وInternal وExternal
  • ✅ External CSS هي الأفضل لفصل الكود وإعادة الاستخدام
  • ✅ المحددات: عنصر (.class) (#id) وشامل (*) ومتعدد ونسب
  • ✅ أولوية CSS (Specificity): ID > Class > Element
