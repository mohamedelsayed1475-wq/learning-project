🔲 شبكة التخطيط - CSS Grid

CSS Grid هو نظام التخطيط ثنائي الأبعاد. مع Flexbox يُكوّنان معاً الثنائي القوي لأي تصميم ويب احترافي!

🔲 ما هو CSS Grid؟

💡 Flexbox = بُعد واحد (صف أو عمود)

💡 Grid = بُعدان (صفوف وأعمدة معاً)

القاعدة: Flexbox للمكونات الصغيرة، Grid لتخطيط الصفحة الكاملة!

/* تفعيل Grid */
.grid-container {
  display: grid;

  /* تعريف الأعمدة */
  grid-template-columns: 200px 200px 200px;
  /* 3 أعمدة بعرض 200px كل منها */

  grid-template-columns: 1fr 1fr 1fr;
  /*
    fr = fraction = جزء
    كل عمود يأخذ جزءاً متساوياً من المساحة المتاحة
    = ثلاثة أعمدة متساوية
  */

  grid-template-columns: repeat(3, 1fr);
  /* نفس الكود السابق لكن أقصر */

  grid-template-columns: 250px 1fr 1fr;
  /* عمود ثابت 250px + عمودان متساويان */

  /* تعريف الصفوف */
  grid-template-rows: 100px 200px 100px;
  /* 3 صفوف بارتفاعات محددة */

  /* المسافات بين الخلايا */
  gap: 20px;           /* مسافة متساوية */
  column-gap: 24px;   /* بين الأعمدة */
  row-gap: 16px;      /* بين الصفوف */
}

📐 وحدة fr والـ minmax

.grid {
  display: grid;

  /* شبكة متجاوبة تلقائياً */
  grid-template-columns: repeat(auto-fill, minmax(280px, 1fr));
  /*
    auto-fill = اصنع أكبر عدد من الأعمدة ممكن
    minmax(280px, 1fr) = كل عمود:
      - لا يقل عن 280px
      - لا يزيد عن 1fr (جزء متاح)
    
    النتيجة: شبكة متجاوبة بدون Media Queries! ✨
    على الموبايل: عمود واحد
    على التابلت: عمودان
    على الحاسوب: ثلاثة أو أكثر
  */
}

🏠 تمتد الخلايا (Grid Span)

/* عنصر يمتد عبر أعمدة أو صفوف متعددة */
.header-item {
  grid-column: 1 / 4;
  /* يبدأ من الخط 1 وينتهي عند الخط 4 = يغطي 3 أعمدة */

  grid-column: 1 / -1;
  /* من البداية حتى النهاية = يغطي كل الأعمدة */

  grid-column: span 2;
  /* يمتد على عمودين من موضعه الحالي */
}

.sidebar {
  grid-row: 1 / 3;
  /* يمتد على صفين (من السطر 1 للسطر 3) */
}

/* مثال: تخطيط صفحة كلاسيكي */
.page-layout {
  display: grid;
  grid-template-columns: 1fr 3fr;
  /* sidebar 1/4 + main content 3/4 */
  grid-template-rows: auto 1fr auto;
  /* header + content + footer */
}

.page-header { grid-column: 1 / -1; }
/* Header يغطي كل العرض */

.page-footer { grid-column: 1 / -1; }
/* Footer يغطي كل العرض */

🎓 ما تعلمناه في هذا الدرس

  • ✅ display: grid يفعّل نظام Grid
  • ✅ grid-template-columns لتعريف الأعمدة
  • ✅ وحدة fr للمساحات النسبية المرنة
  • ✅ repeat() لتكرار الأعمدة/الصفوف
  • ✅ minmax() وauto-fill لشبكة متجاوبة تلقائياً
  • ✅ grid-column وgrid-row وspan للتحكم في امتداد الخلايا
