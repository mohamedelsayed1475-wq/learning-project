💪 التخطيط المرن - Flexbox

Flexbox هو نظام التخطيط الأكثر استخداماً في CSS الحديثة. يجعل محاذاة وترتيب العناصر أمراً بسيطاً وسهلاً!

💡 ما هو Flexbox؟

Flexbox = Flexible Box Layout = نظام التخطيط المرن

يُطبَّق على الحاوية (Container) ويؤثر على أبنائها المباشرين (Items)

/* تفعيل Flexbox على الحاوية */
.container {
  display: flex;
  /*
    هذا السطر الوحيد يحوّل كل الأبناء المباشرين
    ليتراصوا في صف أفقي تلقائياً!
  */
}

📐 خصائص الحاوية (Container)

.flex-container {
  display: flex;

  /* اتجاه المحور الرئيسي */
  flex-direction: row;         /* افتراضي: أفقي من اليسار */
  flex-direction: row-reverse; /* أفقي من اليمين */
  flex-direction: column;      /* رأسي من الأعلى */

  /* هل العناصر تنزل لسطر جديد؟ */
  flex-wrap: nowrap;   /* افتراضي: لا تنزل، تضغط */
  flex-wrap: wrap;     /* تنزل لسطر جديد عند الامتلاء */

  /* المحاذاة على المحور الرئيسي (أفقي افتراضاً) */
  justify-content: flex-start;    /* في البداية (افتراضي) */
  justify-content: flex-end;      /* في النهاية */
  justify-content: center;        /* في الوسط ✅ مفيد جداً */
  justify-content: space-between; /* مساحات بين العناصر */
  justify-content: space-around;  /* مساحات حول العناصر */
  justify-content: space-evenly;  /* مساحات متساوية */

  /* المحاذاة على المحور الثانوي (رأسي افتراضاً) */
  align-items: stretch;      /* يمتد للأعلى والأسفل (افتراضي) */
  align-items: flex-start;  /* يلتصق بالأعلى */
  align-items: flex-end;    /* يلتصق بالأسفل */
  align-items: center;      /* يتوسط رأسياً ✅ مفيد جداً */

  /* مسافة بين العناصر (gap) */
  gap: 20px;           /* مسافة في الصفوف والأعمدة */
  gap: 16px 24px;     /* صفوف ثم أعمدة */
  column-gap: 20px;   /* مسافة بين الأعمدة فقط */
  row-gap: 16px;      /* مسافة بين الصفوف فقط */
}

🎯 خصائص العناصر الأبناء (Items)

.flex-item {
  /* نسبة النمو: كم يأخذ من المساحة الفارغة */
  flex-grow: 0;    /* افتراضي: لا ينمو */
  flex-grow: 1;    /* ينمو ليملأ المساحة الفارغة */
  flex-grow: 2;    /* ينمو ضعف عنصر بـ flex-grow: 1 */

  /* الحجم الافتراضي */
  flex-basis: auto;   /* يعتمد على المحتوى (افتراضي) */
  flex-basis: 200px;  /* حجم أساسي 200px */
  flex-basis: 0;      /* يبدأ من الصفر قبل التوزيع */

  /* اختصار flex */
  flex: 1;          /* = flex-grow: 1; flex-shrink: 1; flex-basis: 0 */
  flex: 1 1 200px;  /* grow, shrink, basis */

  /* ترتيب ظهور العنصر */
  order: 0;  /* افتراضي */
  order: 1;  /* يظهر بعد العناصر ذات order:0 */
  order: -1; /* يظهر أول شيء! */
}

/* ✅ المثال الأكثر شيوعاً: توسيط عنصر */
.center-everything {
  display: flex;
  justify-content: center; /* توسيط أفقي */
  align-items: center;    /* توسيط رأسي */
  /* قبل Flexbox كان التوسيط الرأسي شبه مستحيل! */
}

🎓 ما تعلمناه في هذا الدرس

  • ✅ display: flex يفعّل Flexbox على الحاوية
  • ✅ flex-direction لتحديد اتجاه العناصر
  • ✅ justify-content للمحاذاة على المحور الرئيسي
  • ✅ align-items للمحاذاة على المحور الثانوي
  • ✅ gap لإضافة مسافات بين العناصر
  • ✅ flex-grow وflex-basis لتحديد حجم الأبناء
