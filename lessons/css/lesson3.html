📦 نموذج الصندوق - Box Model

Box Model هو المفهوم الأساسي الأهم في CSS. كل عنصر HTML هو صندوق له أربع طبقات. فهم هذا المفهوم يحل 90% من مشاكل التخطيط!

📦 ما هو Box Model؟

كل عنصر في HTML يُعامَل كصندوق مستطيل يتكون من أربع طبقات من الداخل للخارج:

🟦 Content - المحتوى الفعلي (النص أو الصورة)

🟩 Padding - المسافة الداخلية بين المحتوى والحدود

🟨 Border - الحدود حول العنصر

🟥 Margin - المسافة الخارجية بين العنصر والعناصر الأخرى

/*
  تصور الـ Box Model:
  ┌─────────────────────────────┐  ← Margin (خارجي)
  │   ┌─────────────────────┐   │  ← Border (حدود)
  │   │   ┌─────────────┐   │   │  ← Padding (داخلي)
  │   │   │   Content   │   │   │  ← Content (المحتوى)
  │   │   └─────────────┘   │   │
  │   └─────────────────────┘   │
  └─────────────────────────────┘
*/

.card {
  /* 1️⃣ المحتوى (Content) - يتحدد بـ width وheight */
  width: 300px;   /* عرض منطقة المحتوى */
  height: 200px;  /* ارتفاع منطقة المحتوى */

  /* 2️⃣ الـ Padding - المسافة الداخلية */
  padding: 20px;
  /* 20px على كل الجوانب الأربعة */

  /* 3️⃣ الـ Border - الحدود */
  border: 2px solid #2563eb;
  /*
    2px = سمك الحدود
    solid = نوع الحدود (صلبة)
    #2563eb = لون الحدود
  */

  /* 4️⃣ الـ Margin - المسافة الخارجية */
  margin: 16px;
  /* 16px مسافة من الخارج على كل الجوانب */
}

📐 Padding بالتفصيل

.box {
  /* طريقة 1: نفس القيمة للجوانب الأربعة */
  padding: 20px;
  /* top=20, right=20, bottom=20, left=20 */

  /* طريقة 2: قيمتان - أعلى/أسفل ويمين/يسار */
  padding: 10px 20px;
  /* top=10, right=20, bottom=10, left=20 */

  /* طريقة 3: ثلاث قيم - أعلى، يمين/يسار، أسفل */
  padding: 10px 20px 15px;
  /* top=10, right=20, bottom=15, left=20 */

  /* طريقة 4: أربع قيم - اتجاه عقارب الساعة */
  padding: 10px 20px 15px 25px;
  /* top=10, right=20, bottom=15, left=25 */
  /* تذكر: أعلى، يمين، أسفل، يسار (TRBL) */

  /* طريقة 5: كل جانب منفرداً */
  padding-top: 10px;
  padding-right: 20px;
  padding-bottom: 15px;
  padding-left: 25px;
}

/* نفس القواعد تنطبق على Margin */

🔲 Border بالتفصيل

.bordered {
  /* الصيغة المختصرة: سمك، نوع، لون */
  border: 2px solid #2563eb;

  /* أو كل خاصية منفردة */
  border-width: 2px;
  border-style: solid;    /* خط صلب */
  border-color: #2563eb;

  /*
    أنواع الحدود (border-style):
    solid  = خط صلب ───────
    dashed = خط متقطع - - - -
    dotted = نقاط ............
    double = خط مزدوج ═══════
    none   = لا حدود
  */

  /* حدود جانب معين فقط */
  border-bottom: 3px solid #2563eb;
  /* حدود سفلية فقط - شائع للروابط والعناوين */

  /* تدوير الزوايا */
  border-radius: 8px;
  /* زوايا دائرية بنصف قطر 8px */

  border-radius: 50%;
  /* دائرة كاملة - لو العنصر مربع! */

  border-radius: 20px 0 20px 0;
  /* تدوير غير متماثل - شكل مثير للاهتمام */
}

⚠️ box-sizing: border-box

/*
  المشكلة الشائعة:
  العرض الكلي = width + padding + border + margin
  
  مثال:
  width: 300px + padding: 20px*2 + border: 2px*2
  = 300 + 40 + 4 = 344px إجمالي!
  
  هذا يسبب مشاكل في التخطيط!
*/

/* الحل: box-sizing: border-box */
* {
  box-sizing: border-box;
}
/*
  مع border-box:
  width = المقاس الكلي شامل padding وborder
  
  مثال:
  width: 300px = المقاس النهائي 300px بالضبط!
  (الـ padding والـ border يُطرحان من المحتوى)
  
  ✅ ضعها دائماً في بداية كل مشروع CSS!
*/

/* تقدر تضيف عليها هذا للتوافق الأفضل */
*, *::before, *::after {
  box-sizing: border-box;
}

📏 Width وHeight وoverflow

.container {
  /* عرض ثابت */
  width: 1200px;

  /* عرض نسبي */
  width: 80%;    /* 80% من عرض الأب */
  width: 100vw;  /* 100% من عرض المتصفح */

  /* حد أقصى وأدنى للعرض */
  max-width: 1200px; /* لا يتجاوز 1200px */
  min-width: 320px;  /* لا يقل عن 320px */

  /* توسيط العنصر أفقياً */
  margin: 0 auto;
  /* 0 = margin أعلى/أسفل، auto = margin يمين/يسار تلقائياً متساوية */
}

.text-box {
  width: 200px;
  height: 100px;

  /* ماذا لو المحتوى أكبر من الصندوق؟ */
  overflow: hidden;   /* إخفاء الزائد */
  overflow: scroll;   /* شريط تمرير دائماً */
  overflow: auto;     /* شريط تمرير عند الحاجة فقط ✅ */
  overflow: visible; /* يظهر خارج الصندوق (افتراضي) */
}

🎓 ما تعلمناه في هذا الدرس

  • ✅ Box Model: Content، Padding، Border، Margin من الداخل للخارج
  • ✅ طرق كتابة Padding وMargin: قيمة واحدة أو قيم متعددة
  • ✅ خصائص Border: width وstyle وcolor وradius
  • ✅ box-sizing: border-box لحل مشاكل الأبعاد
  • ✅ Width وHeight وoverflow للتحكم في المساحة
