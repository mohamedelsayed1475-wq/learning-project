📍 التموضع والعرض - Display & Position

تعلم كيف تتحكم في كيفية عرض العناصر وأين تظهر بالضبط في الصفحة! خاصيتا display وposition هما أساس كل تخطيط في CSS.

🖥️ خاصية Display

/*
  display = كيف يتصرف العنصر في تدفق الصفحة
  القيم الأساسية:
*/

/* 1. block - العناصر الكتلية */
div, p, h1 {
  display: block;
  /*
    - يأخذ عرض كامل (100%) من الأب
    - ينزل على سطر جديد
    - يقبل width وheight
    - أمثلة: div, p, h1-h6, section, article
  */
}

/* 2. inline - العناصر المضمنة */
span, a, strong {
  display: inline;
  /*
    - يأخذ عرض محتواه فقط
    - يبقى في نفس السطر
    - لا يقبل width وheight
    - أمثلة: span, a, img, strong, em
  */
}

/* 3. inline-block - الوسط بين الاثنين */
.badge {
  display: inline-block;
  /*
    - يبقى في السطر زي inline
    - لكن يقبل width وheight زي block
    - مفيد للأزرار والشارات والروابط
  */
  width: 100px;
  height: 36px;
}

/* 4. none - إخفاء العنصر كلياً */
.hidden {
  display: none;
  /*
    - يُزيل العنصر من الصفحة كلياً
    - لا يأخذ مساحة
    - مختلف عن visibility: hidden
    - visibility: hidden يُخفي لكن يحتفظ بالمساحة
  */
}

📌 خاصية Position

/* 1. static - الافتراضي */
p {
  position: static;
  /*
    - الوضع الافتراضي لكل العناصر
    - يتبع التدفق الطبيعي للصفحة
    - لا يتأثر بـ top, right, bottom, left
  */
}

/* 2. relative - نسبي */
.shifted {
  position: relative;
  top: 20px;   /* ينزل 20px من موضعه الأصلي */
  left: 10px;  /* ينتقل 10px لليسار */
  /*
    - يتحرك بالنسبة لموضعه الأصلي
    - يحتفظ بمساحته في التدفق الطبيعي
    - الأهم: يصبح نقطة مرجعية للعناصر absolute بداخله!
  */
}

/* 3. absolute - مطلق */
.tooltip {
  position: absolute;
  top: 0;
  right: 0;
  /*
    - يخرج من التدفق الطبيعي (لا يأخذ مساحة)
    - يتموضع بالنسبة لأقرب عنصر أب بـ position غير static
    - لو ما فيش، يتموضع بالنسبة للـ body
    - شائع جداً لـ: tooltips, badges, overlays
    
    مثال: زرار "جديد" فوق صورة المنتج
    الصورة: position relative
    النص "جديد": position absolute, top 10px, left 10px
  */
}

/* 4. fixed - ثابت بالنسبة للشاشة */
.navbar {
  position: fixed;
  top: 0;
  left: 0;
  right: 0;
  z-index: 100;
  /*
    - يبقى ثابتاً عند التمرير (scroll)
    - يتموضع بالنسبة لنافذة المتصفح
    - يخرج من التدفق الطبيعي
    - شائع جداً لـ: navigation bars, chat buttons
    
    z-index = ترتيب الطبقات
    القيمة الأعلى = يظهر فوق كل شيء آخر
  */
}

/* 5. sticky - لاصق */
.sticky-header {
  position: sticky;
  top: 0;
  /*
    - يبدأ كـ relative
    - عند التمرير يصبح كـ fixed عند الحد المحدد
    - يبقى في حاويته (الأب)
    - شائع لـ: جداول بعناوين ثابتة، سكاشن headers
  */
}

🎓 ما تعلمناه في هذا الدرس

  • ✅ display: block وinline وinline-block ونone
  • ✅ position: static وrelative وabsolute وfixed وsticky
  • ✅ الفرق بين display:none وvisibility:hidden
  • ✅ z-index لترتيب طبقات العناصر
  • ✅ top وright وbottom وleft للتحكم في الموضع
