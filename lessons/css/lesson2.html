🎨 الألوان والخلفيات وتنسيق الخطوط

تعلم كيف تتحكم في الألوان بطرق مختلفة (HEX وRGB وHSL)، وكيف تنسّق الخطوط والنصوص لتعطي موقعك مظهراً احترافياً!

🎨 الألوان في CSS

هناك عدة طرق لتحديد الألوان في CSS:

1. الألوان بالاسم (Named Colors)

p {
  color: red;        /* أحمر */
  color: blue;       /* أزرق */
  color: darkgreen;  /* أخضر داكن */
  color: tomato;     /* أحمر طماطم 🍅 */
  color: steelblue;  /* أزرق فولاذي */
}

/*
  ✅ مناسبة للألوان الأساسية البسيطة
  ❌ محدودة - فقط ~140 لون مسمى في CSS
  ❌ الأسماء ليست دقيقة بما يكفي للتصاميم الاحترافية
*/

2. نظام HEX (الأكثر استخداماً)

/*
  HEX = Hexadecimal = نظام عد ست عشري (0-9 ثم A-F)
  الصيغة: #RRGGBB
  RR = قيمة الأحمر (Red) من 00 إلى FF
  GG = قيمة الأخضر (Green) من 00 إلى FF
  BB = قيمة الأزرق (Blue) من 00 إلى FF

  00 = 0 (لا يوجد)
  FF = 255 (أقصى درجة)
*/

h1 {
  color: #FF0000;  /* أحمر كامل: R=255, G=0, B=0 */
  color: #00FF00;  /* أخضر كامل: R=0, G=255, B=0 */
  color: #0000FF;  /* أزرق كامل: R=0, G=0, B=255 */
  color: #000000;  /* أسود: الثلاثة صفر */
  color: #FFFFFF;  /* أبيض: الثلاثة بحدهم */
  color: #2563eb;  /* أزرق جميل - يمكن كتابته بحروف صغيرة */
  color: #1e293b;  /* رمادي داكن جداً */
}

/*
  اختصار HEX: لو كل رقمين متماثلين يمكن اختصارهما
  #FF0000 = #F00
  #FFFFFF = #FFF
  #000000 = #000
  #223366 = #236
*/

3. نظام RGB

/*
  RGB = Red, Green, Blue
  كل قيمة من 0 إلى 255
  rgb(أحمر, أخضر, أزرق)
*/

div {
  color: rgb(255, 0, 0);     /* أحمر كامل */
  color: rgb(37, 99, 235);  /* نفس #2563eb */
  color: rgb(0, 0, 0);       /* أسود */
}

/* RGBA: الـ A = Alpha = الشفافية من 0 إلى 1 */
.overlay {
  background: rgba(0, 0, 0, 0.5);
  /* أسود بشفافية 50% - شائع جداً للطبقات الشفافة */

  background: rgba(37, 99, 235, 0.2);
  /* أزرق بشفافية 20% - تأثير خلفية شفافة */
}

4. نظام HSL (الأكثر بديهية)

/*
  HSL = Hue, Saturation, Lightness
  H = درجة اللون (Hue) من 0 إلى 360 درجة
      0 = أحمر، 120 = أخضر، 240 = أزرق
  S = التشبع (Saturation) من 0% إلى 100%
      0% = رمادي، 100% = لون نقي كامل
  L = الإضاءة (Lightness) من 0% إلى 100%
      0% = أسود، 50% = طبيعي، 100% = أبيض
*/

button {
  background: hsl(220, 90%, 56%);
  /* أزرق: درجة 220، تشبع عالي، إضاءة متوسطة */

  background: hsl(220, 90%, 46%);
  /* نفس اللون لكن أغمق (46% إضاءة) */

  background: hsla(220, 90%, 56%, 0.8);
  /* نفس اللون مع شفافية 80% */
}

/*
  💡 لماذا HSL أفضل للمبدعين؟
  سهل تغيير الإضاءة فقط للحصول على ألوان متناسقة!
  مثال: لو عندك لون أساسي hsl(220, 90%, 56%)
  النسخة الفاتحة: hsl(220, 90%, 80%)
  النسخة الداكنة: hsl(220, 90%, 30%)
  كل اللون من نفس العائلة!
*/

🖌️ الخلفيات (Backgrounds)

.hero-section {
  /* لون خلفية عادي */
  background-color: #1e293b;

  /* تدرج لوني (Gradient) - الأجمل! */
  background: linear-gradient(135deg, #1e293b, #2563eb);
  /*
    linear-gradient = تدرج خطي
    135deg = اتجاه التدرج (135 درجة = قطري)
    #1e293b = اللون الأول (البداية)
    #2563eb = اللون الثاني (النهاية)
  */

  /* صورة خلفية */
  background-image: url('hero.jpg');
  background-size: cover;
  /* cover = الصورة تغطي المساحة كلها مع الحفاظ على النسبة */
  background-position: center;
  /* center = توسيط الصورة */
  background-repeat: no-repeat;
  /* no-repeat = لا تكرر الصورة */
}

/*
  قيم background-size:
  cover  = تغطي الكل (قد تُقطع الصورة)
  contain = تظهر كاملة (قد تظهر فراغات)
  100% 100% = تمدد الصورة لتملأ الكل (قد تتشوه)
*/

📝 تنسيق الخطوط (Typography)

خصائص الخط

body {
  /* نوع الخط */
  font-family: 'Cairo', 'Segoe UI', sans-serif;
  /*
    Cairo = خط عربي جميل من Google Fonts
    'Segoe UI' = خط بديل لو Cairo مش متاح
    sans-serif = خط بلا زخارف كـ fallback أخير
    الفاصلة تعني "استخدم هذا، أو هذا، أو هذا..."
  */

  /* حجم الخط */
  font-size: 16px;     /* الحجم الافتراضي والمريح للقراءة */
  font-size: 1rem;    /* 1rem = حجم الخط الافتراضي للمتصفح (16px) */
  font-size: 1.2em;   /* 1.2 ضعف حجم خط الأب */

  /* وزن الخط (سماكته) */
  font-weight: 400;   /* عادي - normal */
  font-weight: 700;   /* عريض - bold */
  font-weight: 300;   /* رفيع - light */

  /* نمط الخط */
  font-style: normal; /* عادي */
  font-style: italic; /* مائل */

  /* ارتفاع السطر */
  line-height: 1.6;   /* 1.6 مرة حجم الخط = ارتفاع مريح للقراءة */
}

تنسيق النص

.article-text {
  /* محاذاة النص */
  text-align: right;    /* يمين (للعربية) */
  text-align: left;     /* يسار (للإنجليزية) */
  text-align: center;   /* توسيط */
  text-align: justify;  /* ضبط = يمتد عرض السطر بالكامل */

  /* تزيين النص */
  text-decoration: none;         /* إزالة الخط من الروابط */
  text-decoration: underline;    /* خط تحت */
  text-decoration: line-through; /* خط في المنتصف (شطب) */

  /* تحويل حالة الأحرف */
  text-transform: uppercase;  /* كل الأحرف كبيرة */
  text-transform: capitalize; /* أول حرف كبير في كل كلمة */

  /* ظل النص */
  text-shadow: 2px 2px 4px rgba(0,0,0,0.3);
  /*
    2px = إزاحة أفقية
    2px = إزاحة رأسية
    4px = درجة الضبابية (blur)
    rgba(0,0,0,0.3) = لون الظل (أسود شفاف)
  */

  /* تباعد الأحرف */
  letter-spacing: 0.05em;  /* مسافة صغيرة بين الأحرف */

  /* تباعد الكلمات */
  word-spacing: 0.1em;    /* مسافة بين الكلمات */
}

استيراد خطوط Google

/* في ملف HTML داخل <head> */
<!--
<link rel="preconnect" href="https://fonts.googleapis.com">
<link href="https://fonts.googleapis.com/css2?family=Cairo:wght@300;400;700&display=swap" rel="stylesheet">
-->

/* أو في ملف CSS */
@import url('https://fonts.googleapis.com/css2?family=Cairo:wght@300;400;700&display=swap');

/*
  @import = استيراد من مصدر خارجي
  Cairo = اسم الخط العربي الجميل
  wght@300;400;700 = الأوزان التي نريدها
  300 = رفيع، 400 = عادي، 700 = عريض
*/

body {
  font-family: 'Cairo', sans-serif;
  /* الآن يمكن استخدام خط Cairo في كل الصفحة */
}
