🎬 الانتقالات، التحويلات والحركات (Transitions & Animations)

أضف الحياة لصفحتك! تعلم كيف تصمم حركات سلسة وتأثيرات بصرية رائعة تجعل تجربة المستخدم مبهرة باستخدام الانتقالات والحركات.

🔄 الانتقالات (CSS Transitions)

تسمح لك الانتقالات بتغيير قيم الخصائص بسلاسة وبشكل تدريجي (على مدى فترة زمنية محددة)، بدلاً من التغير المفاجئ. 

.btn {
  background-color: #3b82f6; /* اللون الأساسي أزرق */
  color: white;
  padding: 10px 20px;
  border: none;
  border-radius: 5px;
  cursor: pointer;

  /* 🔄 تفعيل الانتقال السلس للون الخلفية */
  transition-property: background-color; /* الخاصية التي نريد تحريكها */
  transition-duration: 0.3s;              /* مدة الحركة (0.3 ثانية) */
  transition-timing-function: ease-in-out; /* سرعة الحركة في البداية والنهاية */
  transition-delay: 0s;                    /* تأخير بدء الحركة */

  /* 🚀 الصيغة المختصرة (Shorthand) - الأكثر استخداماً */
  transition: background-color 0.3s ease-in-out;

  /* لتحريك كافة الخصائص التي تتغير */
  transition: all 0.3s ease;
}

/* تأثير التحويم */
.btn:hover {
  background-color: #1d4ed8; /* سيتغير اللون بسلاسة خلال 0.3 ثانية */
}

📐 التحويلات (CSS Transforms)

تتيح لك خاصية transform إمكانية تغيير شكل العناصر، تدويرها، تكبيرها أو إزاحتها في الفضاء ثنائي الأبعاد (2D) أو ثلاثي الأبعاد (3D). 

.card {
  width: 200px;
  height: 200px;
  background: #ec4899;
  transition: transform 0.4s cubic-bezier(0.175, 0.885, 0.32, 1.275); /* حركة ارتدادية رائعة */
}

.card:hover {
  /* 📐 بعض دوال التحويل الشائعة: */
  
  transform: scale(1.1);          /* تكبير العنصر بنسبة 10% */
  
  transform: rotate(15deg);       /* تدوير العنصر بمقدار 15 درجة باتجاه عقارب الساعة */
  
  transform: translateY(-10px);   /* إزاحة العنصر للأعلى بمقدار 10 بكسل (تأثير رفع ذكي) */
  
  transform: skewX(-10deg);      /* إمالة العنصر أفقياً */

  /* 💫 دمج تحويلات متعددة معاً: */
  transform: translateY(-10px) scale(1.05) rotate(5deg);
}

🎬 الحركات الإبداعية (CSS Animations & @keyframes)

تمنحك الحركات تحكماً كاملاً في الحركة عبر نقاط زمنية متعددة بدون الحاجة لتفاعل المستخدم (مثل التحويم). نقوم بتعريف الحركة باستخدام @keyframes ثم نطبقها على العنصر. 

/* 1️⃣ تعريف الحركة ونقاطها الزمنية */
@keyframes bounceAndColor {
  /* نقطة البداية (0%) */
  0% {
    transform: translateY(0);
    background-color: #3b82f6;
  }
  
  /* منتصف الحركة (50%) */
  50% {
    transform: translateY(-30px); /* القفز للأعلى */
    background-color: #10b981;    /* تغير اللون للأخضر */
  }
  
  /* نهاية الحركة (100%) */
  100% {
    transform: translateY(0);     /* العودة للأسفل */
    background-color: #3b82f6;
  }
}

/* 2️⃣ تطبيق الحركة على العنصر */
.animated-ball {
  width: 60px;
  height: 60px;
  border-radius: 50%;
  
  /* خصائص الحركة المفصلة: */
  animation-name: bounceAndColor;   /* اسم الحركة المعرفة بـ @keyframes */
  animation-duration: 2s;           /* مدة الدورة الواحدة للحركة */
  animation-timing-function: ease;  /* نمط السرعة */
  animation-iteration-count: infinite; /* عدد مرات التكرار (تكرار مستمر) */
  animation-direction: normal;     /* اتجاه الحركة */
  
  /* 🚀 الصيغة المختصرة (Shorthand): */
  animation: bounceAndColor 2s ease infinite;
}

🎓 ما تعلمناه في هذا الدرس

  • ✅ الانتقالات (Transitions) تسمح بتغيير الخصائص بنعومة بدلاً من القفز المفاجئ.
  • ✅ خاصية transform تُغيّر موضع وحجم ودوران العناصر.
  • ✅ ندمج transition مع transform للحصول على حركات تحويم مذهلة.
  • ✅ الحركات (Animations) المعرفة بـ @keyframes تُنشئ حركات معقدة ومستمرة دون تدخل المستخدم.
  • ✅ يمكن تحديد مسار الحركة بدقة بالكامل من 0% حتى 100%.
