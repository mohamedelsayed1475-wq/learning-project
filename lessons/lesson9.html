🗂️ div و span والتنظيم

يُعدّ التنظيم من أهم مهارات كتابة HTML الاحترافي. ستتعلم في هذا الدرس الفرق بين العناصر الكتلية (Block) والمضمّنة (Inline)، وكيف تستخدم div وspan لتنظيم صفحتك بشكل صحيح.

1. الفرق بين Block وInline

كل عنصر HTML ينتمي إلى إحدى المجموعتين الرئيسيتين: Block أو Inline.

📦 Block Elements (عناصر كتلية)

تبدأ دائماً في سطر جديد وتأخذ العرض الكامل المتاح بالصفحة.

<div>, <p>, <h1>-<h6> <ul>, <ol>, <li> <header>, <main>, <footer> <section>, <article>

📝 Inline Elements (عناصر مضمّنة)

تبقى في نفس السطر وتأخذ فقط المساحة التي يحتاجها محتواها.

<span>, <a>, <strong> <em>, <code>, <img> <label>, <button>, <small>
💡 قاعدة عملية مهمة:
لا تضع عنصر Block داخل عنصر Inline! مثلاً لا تضع <div> داخل <span> لأن ذلك يسبب أخطاء في هيكل الصفحة ويُربك المتصفح.

2. وسم <div> — الحاوية الكتلية

<div> هو اختصار لـ Division. وسم حاوية عام ليس له معنى دلالي — يُستخدم فقط لتجميع عناصر مع بعضها وتنسيقها كوحدة واحدة. 

<!-- استخدام div لتجميع محتوى بطاقة -->
<div class="card">
  <h3>عنوان البطاقة</h3>
  <p>محتوى البطاقة هنا...</p>
  <a href="#">اقرأ المزيد</a>
</div>

<!-- div داخل div لبناء تخطيط الصفحة -->
<div class="page-wrapper">
  <div class="sidebar">...</div>
  <div class="content">...</div>
</div>

3. وسم <span> — الحاوية المضمّنة

<span> مثل div تماماً ولكنه Inline — يبقى داخل السطر ويُستخدم لتنسيق جزء محدد من النص دون كسر تدفقه. 

<!-- تلوين كلمة معينة داخل الفقرة -->
<p>
  السماء لونها <span style="color:blue;">أزرق</span>
  والعشب لونه <span style="color:green;">أخضر</span>.
</p>

<!-- إضافة class لكلمة لتنسيقها بـ CSS -->
<p>
  يتعلم <span class="highlight">محمد</span> HTML بكل حماس.
</p>

4. سمة class — التصنيف القابل للتكرار

class هي سمة تُعطي العنصر اسماً للتصنيف، يمكن استخدامه على عناصر متعددة، وتستهدفها في CSS بالنقطة .اسم-الكلاس. 

<!-- HTML: كلاسات على عناصر متعددة -->
<p class="info-text">نص إعلامي أول</p>
<p class="info-text">نص إعلامي ثانٍ</p>
<span class="info-text">نص إعلامي ثالث</span>

/* CSS: استهداف كل العناصر بهذا الكلاس دفعة واحدة */
.info-text {
  color: #007bff;
  font-weight: bold;
}

<!-- عنصر بأكثر من class (نفصل بمسافة) -->
<div class="card featured large">...</div>

5. سمة id — المعرّف الفريد

id معرّف فريد لا يتكرر في نفس الصفحة أبداً — يُستخدم للربط المباشر بعنصر بعينه في CSS أو JavaScript، ويُستهدف بعلامة الهاش #اسم-الـ-id. 

<!-- HTML: تحديد id لعناصر محددة -->
<header id="main-header">...</header>
<section id="hero">...</section>
<footer id="main-footer">...</footer>

/* CSS: استهداف عنصر بعينه */
#main-header {
  background: #007bff;
  position: sticky;
  top: 0;
}

/* JavaScript: الوصول للعنصر */
var header = document.getElementById("main-header");

🔵 class

✅ يتكرر على عناصر عديدة

✅ يمكن تعيين أكثر من class لنفس العنصر

✅ يُستخدم للتنسيق المتكرر

CSS: .اسم-الكلاس { }

🟠 id

⚠️ لا يتكرر — فريد في كل الصفحة

⚠️ عنصر واحد فقط له هذا الـ id

✅ يُستخدم لاستهداف عنصر محدد بدقة

CSS: #اسم-الـ-id { }
💡 متى أستخدم div ومتى أستخدم العناصر الدلالية؟
استخدم العناصر الدلالية (<header>, <main>, <section>...) كلما أمكن لأنها أفضل لـ SEO. استخدم <div> فقط حين لا يوجد عنصر دلالي مناسب للغرض.
