🔍 SEO وتحسين محركات البحث

تريد لموقعك أن يظهر في الصفحة الأولى على جوجل؟ في هذا الدرس سنتعلم كيفية تهيئة صفحات موقعنا لمحركات البحث (SEO) باستخدام أوسمة HTML الوصفية (Meta Tags)، وكيفية هيكلة العناوين ومشاركة الروابط بشكل جذاب على منصات التواصل الاجتماعي.

1. ما هو الـ SEO وكيف تفهم محركات البحث موقعك؟

SEO هي اختصار لـ Search Engine Optimization وتعني "تحسين محركات البحث". هي عملية تهيئة موقعك الإلكتروني لكي يفهمه روبوت محرك البحث (مثل Googlebot) ويقوم بأرشفته وعرضه في النتائج الأولى للباحثين.

تزحف روبوتات البحث داخل كود HTML الخاص بصفحتك، وتبحث عن دلالات وإشارات تفيد بموضوع الصفحة وجودتها. إذا كتبت كوداً منظماً ودلالياً، فستحصل على ترتيب أفضل لدى جوجل.

2. وسوم Meta الرئيسية في الرأس (Head)

توضع وسوم <meta> داخل وسم الـ <head> ولا تظهر للزائر مباشرة، بل توجه للمتصفحات ومحركات البحث. إليك أهم هذه الأوسمة:

meta description (الوصف)
يصف محتوى الصفحة باختصار (بين 150-160 حرفاً). هذا النص هو الذي تراه تحت عنوان موقعك في نتائج بحث جوجل.
meta robots (الروبوتات)
يحدد لروبوتات جوجل إن كان مسموحاً لها أرشفة الصفحة وتتبع روابطها أم لا. مثل: content="index, follow".
meta keywords (الكلمات المفتاحية)
كانت تُستخدم قديماً لوضع كلمات دالة، لكن محركات البحث الحديثة لم تعد تعتمد عليها كثيراً للحد من الحشو العشوائي.
<!-- كود Meta النموذجي لتحسين الأرشفة -->
<meta name="description" content="تعلم لغة HTML من الصفر إلى الاحتراف مع شرح مبسط وعملي باللغة العربية.">
<meta name="robots" content="index, follow">
<meta name="author" content="أحمد المبرمج">

3. بروتوكول Open Graph لمواقع التواصل الاجتماعي

عندما تشارك رابط موقعك على Facebook أو Twitter أو WhatsApp، هل تلاحظ ظهور بطاقة جميلة تحتوي على عنوان وصورة ووصف للموقع؟ هذا يتم باستخدام وسوم Open Graph الخاصة بـ HTML. 

<!-- وسوم Open Graph لتحديد شكل الرابط عند مشاركته -->
<meta property="og:title" content="دورة تعلم HTML5 كاملة بالعربي">
<meta property="og:description" content="ابدأ رحلتك في تطوير الويب الآن مجاناً.">
<meta property="og:image" content="https://example.com/images/cover.jpg">
<meta property="og:url" content="https://example.com/lessons/lesson11.html">
<meta property="og:type" content="article">

4. هيكلية العناوين (Heading Hierarchy) وأثرها على الأرشفة

العناوين ليست مجرد وسيلة لتكبير الخط! العناوين هي الهيكل التنظيمي للمقال.

⚠️ قوانين جوجل الذهبية للعناوين:
1. عنوان H1 واحد فقط: يجب أن تحتوي الصفحة على عنوان <h1> واحد فقط يمثل العنوان الرئيسي للمقال أو الصفحة.
2. التدرج المنطقي: لا تضع <h3> مباشرة تحت <h1>، بل يجب استخدام <h2> كعنصر وسيط لتقسيم الأفكار الرئيسية.

5. البيانات المهيكلة (Schema Markup / JSON-LD)

تُعد البيانات المهيكلة (Structured Data) طريقة متقدمة لإخبار محركات البحث بتفاصيل دقيقة حول محتوى الصفحة (مثل: هل هذه الصفحة مقال، وصفة طبخ، منتج للبيع، أو تقييم كتاب؟).

يتم تضمينها عادةً داخل وسم <script type="application/ld+json"> على شكل كود JSON. 

<!-- مثال على بيانات مهيكلة لمقال تعليمي -->
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "TechArticle",
  "headline": "SEO وتحسين محركات البحث في HTML",
  "description": "تعلم كيفية تهيئة موقعك لمحركات البحث.",
  "author": {
    "@type": "Person",
    "name": "مطور الويب"
  }
}
</script>
💡 سرعة تحميل الصفحة هي جزء من الـ SEO:
كتابة كود HTML نظيف وخالٍ من الأخطاء، مع استدعاء ملفات CSS والـ Scripts بشكل صحيح ومحسّن، يسرّع من استجابة موقعك، وهو عامل ترتيب أساسي في خوارزميات Google الحديثة.
