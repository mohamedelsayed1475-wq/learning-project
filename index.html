❓ أسئلة شائعة

أكتر الأسئلة اللي بيسألها المبتدئين

ما هو الفرق بين HTML و CSS و JavaScript؟

HTML = هيكل الصفحة (زي العظام في الجسم)
CSS = تنسيق وشكل الصفحة (زي الملابس)
JavaScript = التفاعلية والحركة (زي العضلات)

هل HTML لغة برمجة؟

لا! HTML مش لغة برمجة. هي لغة ترميز (Markup Language). الفرق إن البرمجة بتتحكم في المنطق والعمليات، أما HTML فبس بتوصف وتنظم المحتوى.

قديه بياخد وقت أتعلم HTML؟

الأساسيات ممكن تتعلمها في أسبوع أو اتنين لو بتذاكر يومياً. لكن عشان تبقى محترف فعلياً مع CSS و JavaScript محتاج 3-6 شهور. المهم الاستمرار والتطبيق!

محتاج برنامج معين عشان أكتب HTML?

مش محتاج أي برنامج خاص! ممكن تكتب HTML في Notepad العادي. لكن بنفضل تستخدم VS Code (مجاني) لأنه بيساعدك كتير.

إيه بعد HTML؟

بعد HTML تعلم CSS للتنسيق، بعدين JavaScript للتفاعلية، وبعدين تختار: Front-end Framework زي React أو Vue، أو Back-end زي Node.js أو Python.
