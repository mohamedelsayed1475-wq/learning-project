📋 المرجع السريع الشامل لتطوير الويب

كل الوسوم والسمات والخصائص والبرمجيات في مكان واحد — احتفظ بهذه الصفحة مرجعاً دائماً لك أثناء البرمجة.

🏗️ هيكل الصفحة الأساسي

<!DOCTYPE html>
إعلان نوع المستند للمتصفح بأنه HTML5.
<html lang="ar">
جذر الصفحة الرئيسي الذي يغلف كافة العناصر، مع سمة تحديد لغة المحتوى.
<head>
رأس الصفحة ويحمل البيانات الوصفية والإعدادات غير المرئية للمستخدمين.
<meta charset="UTF-8">
ترميز الحروف العالمي لضمان سلامة قراءة اللغة العربية والأحرف الخاصة.
<meta name="viewport" ...>
إعدادات شاشة العرض لجعل صفحات الويب تتجاوب وتتوافق مع مقاسات الموبايل.
<title>
عنوان التبويب العلوي للصفحة في المتصفح.
<body>
جسم الصفحة الذي يضم كافة العناصر المرئية كالفقرات والعناوين والصور.

📝 العناوين

<h1>
العنوان الرئيسي والأهم بالصفحة — يجب استخدامه مرة واحدة فقط.
<h2>
عنوان فرعي من المستوى الثاني (لأقسام الصفحة الكبيرة).
<h3>
عنوان فرعي من المستوى الثالث (للأقسام الفرعية).
<h4> — <h6>
عناوين فرعية أصغر حجماً بالتدريج.
💡 h1 مهم جداً لأرشفة محركات البحث (SEO) — اكتب واحداً فقط لكل صفحة.

✍️ النصوص والتنسيق

<p>
فقرة نصية عادية.
<strong>
نص عريض ومهم لمحركات البحث وللزائر.
<em>
نص مائل للتأكيد اللفظي.
<mark>
نص مظلل بخلفية صفراء لجذب الانتباه.
<small>
نص أصغر حجماً (يستخدم عادة لحقوق الملكية الفكرية أو الشروط والأحكام).
<del>
نص يتوسطه خط مائل يوضح حذفه أو إلغاء العرض.
<code>
عرض نص مخصص كشفرة برمجية inline بخط ثابت.
<br>
وسم فارغ لكسر السطر والنزول لسطر جديد.
<hr>
خط أفقي فاصل للفصل بين المواضيع المختلفة.

🔗 الروابط

<a href="URL">
وسم الرابط ومساره href يحدد الوجهة والهدف.
target="_blank"
سمة تفتح الرابط الخارجي في تبويب جديد للحفاظ على بقاء الزائر بموقعك.
href="page.html"
رابط نسبي يشير لصفحة داخلية بالمشروع.
href="#section-id"
رابط قفز داخلي ينتقل مباشرة لـ id عنصر في نفس الصفحة.
href="mailto:x@x.com"
رابط لفتح تطبيق البريد وإرسال رسالة مباشرة.
href="tel:+201234567"
رابط هاتف يتيح للزائر الاتصال مباشرة عند نقره بالهاتف.

🖼️ الصور

<img src="..." alt="...">
وسم عرض الصورة — وسم فارغ بلا حاجة لإغلاق.
src
سمة تحديد مسار الصورة أو رابطها الإلكتروني.
alt
سمة النص البديل لوصف الصورة لمحركات البحث وقارئات المكفوفين.
width / height
أبعاد الصورة للعرض والارتفاع بالبكسل.
<figure> + <figcaption>
تجميع الصورة وعنوانها الوصفي المكتوب أسفلها معاً بدلالة واضحة.

📋 القوائم

<ul>
قائمة بنقاط غير مرتبة.
<ol>
قائمة مرقمة مرتبة تسلسلياً.
<li>
عنصر القائمة الفردي ويوضع داخل ul أو ol.
ol type="A"
تعديل نوع الترقيم ليصبح بالحروف الأبجدية الإنجليزية.
ol type="I"
تعديل نوع الترقيم ليصبح بالأرقام الرومانية.
<dl> / <dt> / <dd>
قائمة تعريفات ومصطلحات وشروحات مرافقة لها.

📊 الجداول

<table>
الحاوية الأساسية للجدول والبيانات.
<thead>
رأس الجدول ويجمع الصفوف الخاصة بالعناوين.
<tbody>
جسم الجدول ويجمع صفوف البيانات الفعلية المكررة.
<tfoot>
تذييل الجدول ويستخدم لعرض الإجماليات أو الخلاصات.
<tr>
وسم الصف الأفقي بالجدول.
<th>
خلية عنوان العمود (تظهر غامقة ومتوسطة تلقائياً).
<td>
خلية بيانات عادية بالجدول.
colspan="2"
دمج الخلايا أفقياً لعرض عمودين بمساحة عمود واحد مدمج.
rowspan="2"
دمج الخلايا عمودياً لعرض صفين بمساحة صف واحد مدمج.

📝 النماذج

<form action method>
الحاوية الأساسية للنموذج والمدخلات.
<input type="text">
حقل إدخال نصوص عادي.
<input type="email">
حقل إدخال بريد إلكتروني يتحقق المتصفح من صحة ترميزه تلقائياً.
<input type="password">
حقل إدخال كلمة مرور يخفي الأحرف المدخلة لأسباب أمنية.
<input type="checkbox">
مربع اختيار يتيح تحديد خيارات متعددة.
<input type="radio">
زر راديو دائري يتيح خياراً واحداً فقط للمجموعة التي تملك نفس سمة name.
<textarea>
منطقة كتابة نص طويلة متعددة الأسطر.
<select> + <option>
القائمة المنسدلة والخيارات التابعة لها.
<label for="id">
تسمية توضيحية لحقل الإدخال المقابل لزيادة سهولة النقر.
<button type="submit">
زر إرسال بيانات النموذج للخادم.
required
سمة تمنع إرسال النموذج إذا ترك المستخدم الحقل فارغاً.
placeholder
نص تلميحي باهت يظهر داخل الحقل ويختفي عند الكتابة.

🏗️ البنية الدلالية (HTML5)

<header>
رأس وتوثيق الصفحة أو القسم الخاص.
<nav>
تجميع روابط التصفح وقائمة التنقل الأساسية للموقع.
<main>
المحتوى الفريد والوحيد الأساسي بالصفحة.
<section>
تقسيم الصفحة لأقسام وفصول مستقلة وذات موضوع مشترك.
<article>
كتلة محتوى مستقلة بذاتها ومفهومة دون الاعتماد على البقية.
<aside>
شريط جانبي أو معلومات فرعية ملحقة للموضوع الرئيسي.
<footer>
تذييل الصفحة ومعلومات الاتصال وحقوق النشر.
<figure>
تغليف للرسوم البيانية والملفات التوضيحية والصور.
<time datetime="...">
تمثيل التواريخ والأوقات لزيادة فهم محركات البحث.
<address>
بيانات الاتصال والتواصل بالكاتب أو الشركة.

🎯 div و span و class و id

<div>
حاوية كتلية عامة تنزل لسطر جديد لتجميع العناصر.
<span>
حاوية مضمنة تبقى بنفس السطر وتلائم حجم محتواها الداخلي.
class="name"
فئة لتصنيف العناصر، يمكن تكرارها، ويتم استهدافها بـ CSS بالنقطة (.name).
id="name"
معرّف فريد لا يجوز تكراره بالصفحة، ويتم استهدافه بـ CSS بالهاش (#name).
⚡ Block ينشئ سطراً جديداً ويأخذ العرض كله — Inline يبقى بالسطر بمساحة محتواه.

✨ وسوم مفيدة أخرى

<pre>
نص يحافظ على المسافات والأسطر كما كُتبت في الكود تماماً.
<blockquote>
اقتباس نصي طويل من مصدر خارجي يتم تهميشه جانباً.
<abbr title="...">
اختصار يظهر معناه التفصيلي كتلميح عند تحريك الماوس فوقه.
<details> + <summary>
صندوق معلومات قابل للفتح والطي بلمسة واحدة دون جافاسكريبت.
<iframe src="...">
تضمين صفحة ويب خارجية بالكامل داخل صفحتك (مثل الخرائط والفيديوهات).
<video src="...">
عرض وتشغيل ملفات الفيديو مباشرة بالصفحة.
<audio src="...">
تشغيل الملفات الصوتية مباشرة بالصفحة.

⚙️ السمات العامة (Global Attributes)

class
تحديد فئة تصنيف العنصر لاستدعاء تنسيق CSS.
id
تحديد معرّف فريد لاستهدافه بـ CSS أو JavaScript.
style="..."
كتابة أكواد CSS وتنسيقات مباشرة للعنصر.
title="..."
نص تلميحي يظهر عند بقاء الماوس فوق العنصر ثانية.
lang="ar"
تحديد لغة العنصر المحددة بشكل خاص.
dir="rtl"
تحديد اتجاه المحتوى للعنصر بشكل خاص (rtl = يمين لليسار).
hidden
إخفاء العنصر بالكامل من واجهة العرض بالمتصفح.
tabindex
تحديد ترتيب تنقل مؤشر لوحة المفاتيح عبر زر Tab.
aria-label
إعطاء وصف وتسمية للعنصر مخصصة لقارئات المكفوفين الصوتية.

🎨 المحددات وطرق الكتابة

طرق الكتابة
1. Inline: مباشرة في وسم HTML عبر سمة style="...".
2. Internal: داخل وسم <style> في الـ head.
3. External: في ملف مستقل .css يتم ربطه بـ <link rel="stylesheet" ...>.
المحددات (Selectors)
- محدد العنصر: h1 { } لتحديد كافة وسوم h1.
- محدد الكلاس: .box { } لتحديد العناصر ذات class="box".
- محدد الـ ID: #header { } لتحديد العنصر ذي id="header".
- المحدد العام: * { } لتحديد كافة عناصر الصفحة.
أولوية التنسيق
الـ ID هو الأعلى أولوية، يليه الكلاس، ثم العنصر. للتخطي الإجباري نستخدم !important.

🎨 الألوان والخلفيات

تعريف الألوان
- الأسماء: red, blue, gold.
- الـ HEX: #ff0000 (أحمر).
- الـ RGB: rgb(255, 0, 0).
- الـ HSL: hsl(0, 100%, 50%) (درجة اللون، التشبع، الإضاءة).
الخلفيات (Backgrounds)
- background-color: لتحديد لون الخلفية.
- background-image: لتحديد صورة خلفية url('image.jpg').
- background-size: تحديد مقاس الصورة (مثل cover لتغطية الحاوية).
- background-repeat: تكرار الصورة أو منعه no-repeat.
التدرجات والشفافية
- linear-gradient(direction, color1, color2): تدرج لوني خطي.
- opacity: تحديد شفافية العنصر بالكامل (من 0 لـ 1).

✍️ النصوص والخطوط

خصائص الخطوط
- font-size: حجم الخط (بـ px أو rem أو em).
- font-family: نوع الخط المستعمل (مثل Arial, Tahoma).
- font-weight: سمك الخط (مثل bold أو قيم رقمية كـ 400, 700).
تنسيق النصوص
- color: لون الخط.
- text-align: محاذاة النص (right, left, center, justify).
- text-decoration: زخرفة النص (مثل underline للسطر السفلي أو none لإزالة خطوط الروابط).
- line-height: ارتفاع السطر لتحديد المسافة الرأسية بين السطور.

📦 نموذج الصندوق (Box Model)

مكونات الصندوق
1. Content: المحتوى الفعلي (نص أو صورة).
2. Padding: الهامش الداخلي (بين المحتوى والحدود).
3. Border: حافة وإطار العنصر.
4. Margin: الهامش الخارجي الفاصل بين العنصر والعناصر الأخرى.
الحدود (Border)
border: 2px solid black; (السمك، النوع، اللون). ويستخدم border-radius لعمل حواف دائرية.
box-sizing: border-box
خاصية مهمة جداً تجعل أبعاد العنصر (Width/Height) تشمل الهوامش الداخلية والحدود لسهولة الحساب ومنع تمدد العناصر.

📌 التموضع والعرض

العرض (Display)
- block: يأخذ السطر كاملاً (مثل div, p).
- inline: يأخذ مساحة محتواه فقط ولا ينشئ سطراً جديداً (مثل span, a).
- inline-block: كالعناصر السطرية ولكنه يقبل تحديد الطول والعرض والهوامش.
- none: إخفاء العنصر تماماً من العرض.
التموضع (Position)
- static: طبيعي وتلقائي.
- relative: نسبي لموقعه الأصلي.
- absolute: مطلق لحاوية العنصر الأب ذات التموضع النسبي.
- fixed: ثابت بالنسبة للشاشة لا يتحرك مع التمرير.
- sticky: لاصق، يتحرك مع الصفحة ثم يثبت في موقع معين.
ترتيب الطبقات (z-index)
يحدد العنصر الأكثر ارتفاعاً وتداخلاً بالأعلى (يقبل أرقام سالبة وموجبة، ويعمل فقط مع العناصر الموضعة).

↔️ Flexbox & Grid

التخطيط المرن (Flexbox)
- display: flex;: تفعيل النظام للحاوي.
- flex-direction: اتجاه العناصر (row أفقياً أو column عمودياً).
- justify-content: توزيع المحتوى أفقياً (space-between, center, flex-start).
- align-items: محاذاة العناصر عمودياً (center, stretch).
تخطيط الشبكة (Grid)
- display: grid;: تفعيل النظام للشبكة.
- grid-template-columns: تحديد الأعمدة (مثل repeat(3, 1fr)).
- grid-template-rows: تحديد الصفوف.
- gap: المسافة بين الخلايا في التخطيط.

📱 التصميم المتجاوب

الاستعلامات الإعلامية (Media Queries)
تستخدم لتطبيق تنسيقات مخصصة عند مقاسات شاشات محددة:
@media (max-width: 768px) { ... } (يطبق على شاشات الجوال التي يقل عرضها عن 768 بكسل).
الوحدات النسبية (Relative Units)
- rem: نسبة لحجم الخط في العنصر الجذر (html).
- em: نسبة لحجم خط العنصر الأب الحالي.
- %: نسبة لعرض العنصر الأب.
- vw / vh: نسبة مئوية من عرض / طول شاشة العرض (Viewport).

🎬 الحركات والتأثيرات

الانتقالات (Transitions)
تغيير الخصائص بنعومة وسلاسة عند حدوث تغيير (مثل hover):
transition: all 0.3s ease; (الخصائص، المدة، سرعة الحركة).
التحولات (Transforms)
تغيير أبعاد أو موقع العنصر هندسياً:
- scale(1.2): تكبير حجم العنصر.
- rotate(45deg): تدوير العنصر 45 درجة.
- translate(x, y): إزاحة العنصر في الاتجاهين.
الحركات (Animations)
تعريف حركة مخصصة بـ @keyframes my-move { ... } ثم ربطها بالعنصر عبر خاصية animation مع تحديد المدة والتكرار.

⚡ أساسيات جافاسكريبت والمخرجات

إدراج الكود
1. داخلي: داخل وسم <script> ... </script>.
2. خارجي: في ملف مستقل .js يربط عبر <script src="script.js"></script>.
إخراج البيانات
- console.log(value): طباعة البيانات في كونسول المتصفح (مهم للمطور).
- alert(message): إظهار نافذة منبثقة للزائر.
- document.write(text): الكتابة مباشرة في الصفحة.
التعليقات (Comments)
- // تعليق سطر واحد.
- /* تعليق متعدد الأسطر */.

🔢 المتغيرات وأنواع البيانات

تعريف المتغيرات
- let age = 25;: متغير يمكن تعديل قيمته لاحقاً (محدود النطاق).
- const PI = 3.14;: ثابت لا يمكن تعديل قيمته بعد تعريفه.
- var name = "Ali";: الطريقة القديمة (غير مفضلة حالياً بسبب النطاق).
أنواع البيانات (Data Types)
- String: نصوص "Hello".
- Number: أرقام صحيحة وعشرية 42, 3.14.
- Boolean: منطقي true أو false.
- Array: مصفوفة [1, 2, 3].
- Object: كائن يحتوي على خصائص { name: 'Ali', age: 25 }.
- Undefined: متغير تم تعريفه دون إعطائه قيمة.
- Null: قيمة فارغة ومحددة عمداً.

➕ العمليات والمنطق

العمليات الحسابية
- + (جمع / دمج نصوص).
- - (طرح)، * (ضرب)، / (قسمة).
- % (باقي القسمة).
- ++ (زيادة بمقدار 1)، -- (نقص بمقدار 1).
المقارنات
- == (تساوي القيمة فقط).
- === (تساوي القيمة ونوع البيانات معاً - مفضل دائماً).
- != (لا يساوي)، !== (لا يساوي تماماً القيمة والنوع).
- >, <, >=, <= (أكبر، أصغر، أكبر من أو يساوي، أصغر من أو يساوي).
العمليات المنطقية
- &&: و (AND) - يجب تحقق الشرطين معاً.
- ||: أو (OR) - يكفي تحقق أحد الشرطين.
- !: ليس (NOT) - نفي الشرط وعكس قيمته.

🔄 الشرط والتكرار

الجمل الشرطية
- if (condition) { } else if { } else { }.
- الشرط المختصر (Ternary Operator):
let result = (score > 50) ? "ناجح" : "راسب";
التكرار (Loops)
- For Loop: تكرار بعدد معلوم من المرات:
for (let i = 0; i < 5; i++) { ... }
- While Loop: تكرار مستمر طالما الشرط متحقق:
while (condition) { ... }
التحكم في المسار
- break: إيقاف الحلقة والتكرار فوراً والخروج.
- continue: تخطي الدورة الحالية والانتقال للدورة التالية مباشرة.

🧩 الدوال (Functions)

تعريف دالة عادية
function sayHello(name) {
  return "مرحباً " + name;
}
- name: معامل (Parameter) يتم تمريره للدالة.
- return: القيمة المرجعة التي تعيدها الدالة عند استدعائها.
دوال السهم (Arrow Functions)
طريقة حديثة ومختصرة لتعريف الدوال:
const sayHello = (name) => "مرحباً " + name;
النطاق (Scope)
- Global Scope: المتغيرات المعرفة بالخارج متاحة في كل مكان.
- Local Scope: المتغيرات المعرفة داخل دالة أو كتلة {} لا يمكن الوصول إليها من الخارج.

🗂️ المصفوفات والكائنات

المصفوفات (Arrays)
قائمة مرتبة من القيم:
let fruits = ["تفاح", "موز", "برتقال"];
- fruits[0]: الوصول للعنصر الأول (تفاح).
- fruits.push("خوخ"): إضافة عنصر جديد للنهاية.
- fruits.length: معرفة عدد عناصر المصفوفة.
الكائنات (Objects)
مجموعة من الخصائص الممثلة بـ (مفتاح : قيمة):
let user = { name: "خالد", age: 30 };
- user.name: الوصول لقيمة الاسم (خالد).
- user.age = 31;: تعديل قيمة العمر.

🌳 التحكم بالصفحة والأحداث (DOM)

اختيار العناصر
- document.getElementById("my-id"): تحديد عنصر بالـ ID.
- document.querySelector(".my-class"): تحديد أول عنصر يطابق المحدد (كلاس أو وسم أو ID).
- document.querySelectorAll("p"): تحديد كافة العناصر المطابقة.
تعديل العناصر
- element.textContent = "نص جديد";: تعديل النص الداخلي.
- element.innerHTML = "<strong>نص عريض</strong>";: تعديل محتوى HTML.
- element.style.color = "red";: تعديل ستايل الـ CSS مباشرة.
التعامل مع الأحداث
جعل الصفحة تتفاعل مع نقرات وأفعال المستخدم:
element.addEventListener("click", function() {
  alert("تم الضغط!");
});

💾 التخزين المحلي (LocalStorage)

حفظ وجلب البيانات
لحفظ بيانات دائمة في متصفح المستخدم:
- localStorage.setItem("key", "value");: حفظ قيمة.
- let val = localStorage.getItem("key");: جلب قيمة.
- localStorage.removeItem("key");: حذف قيمة محددة.
- localStorage.clear();: مسح كافة البيانات المخزنة.
التعامل مع البيانات المعقدة
لأن التخزين المحلي يدعم النصوص فقط، نستخدم JSON لتحويل البيانات:
- الحفظ: localStorage.setItem("user", JSON.stringify(obj)); (يحول الكائن لنص).
- الجلب: let obj = JSON.parse(localStorage.getItem("user")); (يعيد النص لكائن).
